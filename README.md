# 555手风琴
部分一：电池+稳压器+电源输出和电源正常指示灯
部分二：用555定时器作为音调发生器来生成方波，接着我们会获取那个方波并将其放大，将其通过放大器，再输出到一个小扬声器
我们准备使用一块9V电池，9V电池的问题在于它们无法提供很大的电流，它们的电压较高，但电流较低（一个9V电池里面通常是六节四号电池串联起来，因为是串联的，你能够获得电压，但电流却不会增加。得到的只是9V电池的电流驱动力，也就几百毫安，然后很快，电池就会发热，电量随之减少）关键是，这东西是用9V电池供电的，它不可能永远工作下去，因为我们要用放大器和扬声器，会消耗大量电流。不过，如果你用的是现代的锂离子9V电池，那就能持久些。接下来是开关，如果可能的话，在你的设计中加入开关。稳压器不是那种低效率的7805线性稳压器，是低压差稳压器LDO（效率更高，输入电压和稳压电压之间的压差只需要很小，只需几百毫伏的压差就能维持稳压，即输入电压比稳压电压高出几百毫伏），这意味着，当电池电量逐渐耗尽时，电池电压从9V讲到8V、7V、6V，甚至5.1V时，我们仍能稳定输出5V电压，即使我们把电池用到濒临耗尽
