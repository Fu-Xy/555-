# 555手风琴
部分一：电池+稳压器+电源输出和电源正常指示灯
部分二：用555定时器作为音调发生器来生成方波，接着我们会获取那个方波并将其放大，将其通过放大器，再输出到一个小扬声器
我们准备使用一块9V电池，9V电池的问题在于它们无法提供很大的电流，它们的电压较高，但电流较低（一个9V电池里面通常是六节四号电池串联起来，因为是串联的，你能够获得电压，但电流却不会增加。得到的只是9V电池的电流驱动力，也就几百毫安，然后很快，电池就会发热，电量随之减少）关键是，这东西是用9V电池供电的，它不可能永远工作下去，因为我们要用放大器和扬声器，会消耗大量电流。不过，如果你用的是现代的锂离子9V电池，那就能持久些。接下来是开关，如果可能的话，在你的设计中加入开关。稳压器不是那种低效率的7805线性稳压器，是低压差稳压器LDO（效率更高，输入电压和稳压电压之间的压差只需要很小，只需几百毫伏的压差就能维持稳压，即输入电压比稳压电压高出几百毫伏），这意味着，当电池电量逐渐耗尽时，电池电压从9V讲到8V、7V、6V，甚至5.1V时，我们仍能稳定输出5V电压，即使我们把电池用到濒临耗尽。如果我们使用更老旧的443芯片，我们就需要输入电压比稳压电压高出好几伏。这意味着当稳压目标是5伏时，输入端至少有7V电压。总之，要使用LDO ，使用现代稳压器。如果你要用这种线性稳压器而不是开关稳压器，务必确保它是LDO，而且是高效的。输入端的电容和输出端的几个电容，用于储能和旁路滤波。有了电源指示灯就知道设备是否已经开机。有几个小铜垫测试点，这样我们可以把探针放上去检查，比如电压、信号等等。这些都非常有用。
电源是在VCC处生成的。我们把电源接到555定时器也就是我们的时钟发生器，我们希望使用能够在较低电压下工作的555定时器，因为当电源电压降低并失去稳压时到达某个点，稳压器就会停止工作，电池将直接通过稳压器供电，不再进行稳压即使电压降到4V或3V，这个产品仍将继续运行，因为这个时钟发生器是CMOS的，它能降到大约3V还能继续运行。如果一个555定时器在正常的不稳定或稳定模式下，比如作为振荡器使用，我们知道的是有一个模拟此处频率的模型方程（T、f、D占空比）。理想情况下，我们希望这里的占空比能到达50%，这样信号高电平持续时间就能和低电平一样长
