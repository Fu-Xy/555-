# 555手风琴
部分一：电池+稳压器+电源输出和电源正常指示灯
部分二：用555定时器作为音调发生器来生成方波，接着我们会获取那个方波并将其放大，将其通过放大器，再输出到一个小扬声器
我们准备使用一块9V电池，9V电池的问题在于它们无法提供很大的电流，它们的电压较高，但电流较低（一个9V电池里面通常是六节四号电池串联起来，因为是串联的，你能够获得电压，但电流却不会增加。得到的只是9V电池的电流驱动力，也就几百毫安，然后很快，电池就会发热，电量随之减少）关键是，这东西是用9V电池供电的，它不可能永远工作下去，因为我们要用放大器和扬声器，会消耗大量电流。不过，如果你用的是现代的锂离子9V电池，那就能持久些。接下来是开关，如果可能的话，在你的设计中加入开关。稳压器不是那种低效率的7805线性稳压器，是低压差稳压器LDO（效率更高，输入电压和稳压电压之间的压差只需要很小，只需几百毫伏的压差就能维持稳压，即输入电压比稳压电压高出几百毫伏），这意味着，当电池电量逐渐耗尽时，电池电压从9V讲到8V、7V、6V，甚至5.1V时，我们仍能稳定输出5V电压，即使我们把电池用到濒临耗尽。如果我们使用更老旧的443芯片，我们就需要输入电压比稳压电压高出好几伏。这意味着当稳压目标是5伏时，输入端至少有7V电压。总之，要使用LDO ，使用现代稳压器。如果你要用这种线性稳压器而不是开关稳压器，务必确保它是LDO，而且是高效的。输入端的电容和输出端的几个电容，用于储能和旁路滤波。有了电源指示灯就知道设备是否已经开机。有几个小铜垫测试点，这样我们可以把探针放上去检查，比如电压、信号等等。这些都非常有用。
电源是在VCC处生成的。我们把电源接到555定时器也就是我们的时钟发生器，我们希望使用能够在较低电压下工作的555定时器，因为当电源电压降低并失去稳压时到达某个点，稳压器就会停止工作，电池将直接通过稳压器供电，不再进行稳压即使电压降到4V或3V，这个产品仍将继续运行，因为这个时钟发生器是CMOS的，它能降到大约3V还能继续运行。如果一个555定时器在正常的不稳定或稳定模式下，比如作为振荡器使用，我们知道的是有一个模拟此处频率的模型方程（T、f、D占空比）。理想情况下，我们希望这里的占空比能到达50%，这样信号高电平持续时间就能和低电平一样长，这样就能让声音听起来好一些，声音的音色会更好。我们这里用的是方波，方波实际上是无限多个正弦波的叠加，所以他不会是纯正弦波。有趣的是，如果你用一个正弦波然后把它通过这种小扬声器播放，它听起来特别单薄。但如果换成方波，声音就会更饱满。因为你得到了基波正弦波，无论它的频率是多少，比如440 HZ,然后你还有它的谐波，因为方波是由多个正弦波叠加而成的，声音听起来就会更饱满。如何获得这些不同的频率呢？我们要做的就是放置精准的精密电阻器在RA、RB的电路中，RA是固定的，使用开关，每一个开关都会接入一个电阻作为RB。无论这些值是多少，都会决定我们最终的音调，当我们切换时可以按下开关。没人说不能同时按下这些开关，这样电阻就会以并联方式工作呢，会得到不同的音调。单独按下去永远不会得到多个音调，而且也不会损坏设备。当信号最终产生后，它会像往常一样从out引脚输出，我们这里还有另一个测试点，TPSIg,也就是信号测试点
然后它将通过电容耦合，进入我们的一个放大器，基于晶体管的电流放大器。我们这有一个大型电容器作为储能电容器，因为我们会抽取大量电流，当这个设备启动时，会产生大量电流，然后我们有10K和1K的电阻用来设置我们的偏置点，从电源到集电极有一个二极管，它是我们的续流二极管，因为这里有感性负载，我们不想烧坏晶体管。

添加电压调节器的目的是我们想用一个9V电池，然后将其降压调节到5V，我们希望这个稳压效果保持良好和稳定。LDO低压差线性稳压器，只需要很小的压差在稳定输出电压和输入电压之间。我们在输入端有一个电容，既然我们是用电池供电，我们就不需要任何的高频滤波了，所以没有类似于0.1微法的小电容了。通常来说，如果你使用其他电源，你会希望在这里进行滤波，你可能需要几个电容器，一个大型电容器用于平滑，另一个电容器用于滤波，很多时候你会加入一个电感器或铁氧体磁珠用于抑制非常高频的噪声。但这种情况下是从电池取电，我们会放一个22到100微法的大容量电容器，这样做的原因是我不想一直从电池取电，如果能从电容器中获取电荷那将减轻电池的负担。
